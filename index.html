严谨规则

先判断信号流向，再判断 AI/AO/DI/DO，最后匹配型号。没有资料来源或置信度不足时，只给候选和补问，不直接下最终结论。

选型逻辑
  1. 现场设备 → PLC/DCS：AI 或 DI。
  2. PLC/DCS → 现场设备：AO 或 DO。
  3. 一入一出 = 单通道。
  4. 一入两出 = 一分二。
  5. 两入两出 = 双通道。
  6. T 系列优先；客户强调便宜时补 C 系列候选。
